前端本地使用方法
1.下载node.js
https://nodejs.org/en
下载后在terminal里输入node -v如出现版本号则成功

2.启动项目
在terminal里cd到my-app文件夹
先输入npm install安装所有依赖到本文件夹
再输入npm run dev启动本地服务器，打开浏览器输入http://localhost:3000即可访问。
